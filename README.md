## Description
- 데이터를 분석하고 인사이트를 발굴하여 군집분석, 연관분석, 분류 모델링을 통한 수익 증대 해결방안을 도출
- VIP 고객을 예측하여 품질과 가격이 높은 상품을 추천 및 묶음 상품 판매
- 연령,성별 등 고객의 정보를 고려한 상품을 추천하여 구매 욕구를 일으켜 마케팅 전략 최적화
